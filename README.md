# typedoc theme pikachu

> fork from [typedoc-default-themes](https://github.com/TypeStrong/typedoc-default-themes)

## usage

```bash
yarn add typedoc-pikachu-themes
typedoc --theme node_modules/typedoc-pikachu-themes/bin
```
